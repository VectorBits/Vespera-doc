# 开始使用

这份文档提供一条从“安装配置”到“完成一次扫描”的最短路径，并补齐常用功能的参考说明。

常见用法：
- **下载 + 批量扫描（推荐）**：先把合约写入 MySQL，再从 DB 扫描
- **少量目标直扫**：从文件或单地址直接扫描

推荐按以下顺序阅读：
1. [安装与准备](install.md)
2. [配置文件 settings.yaml](configuration.md)
3. [快速开始](quick-start.md)
4. 按需深入：
   - [Mode 1：定向扫描](mode1.md)
   - [Mode 2：混合扫描](mode2.md)
   - [CLI 参数](cli.md)
   - [报告与日志](report-and-logs.md)
